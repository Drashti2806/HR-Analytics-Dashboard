📊 HR Analytics Dashboard – Employee Attrition & Workforce Insights
📌 Overview

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition and workforce trends. It helps organizations understand key factors influencing employee turnover and improve decision-making.

🎯 Key Objectives
Analyze employee attrition patterns

Identify key factors affecting employee retention

Provide insights on workforce demographics

Support HR teams with data-driven decisions

📈 Dashboard Highlights

🔹 Key Metrics

🔹 Attrition Analysis

    By Gender
    By Education
    By Age Group
    By Salary Slab
    By Job Role
    By Years at Company
🛠️ Tools & Technologies :

Power BI Desktop – Data visualization

DAX (Data Analysis Expressions) – Calculations & measures

Data Modeling – Relationships & insights

💡 Key Insights

Employees with lower salaries tend to leave more

Mid-age professionals (26–35) show highest attrition

Sales-related roles have higher turnover

Early tenure employees are more likely to leave
